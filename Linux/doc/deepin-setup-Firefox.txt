下载最新版的Firefox:http://www.firefox.com.cn/,点击下载，默认下载到当前用户的Download文件夹下
使用如下命令将下载后的安装包解压到/opt目录下

cd ~/Download
tar -xvf  Firefox-latest-x86_64.tar.bz2 
sudo mv firefox /opt/firefox


新增Desktop链接
cd /opt/firefox，找到firefox可执行文件，右键发送到桌面快捷方式。再将桌面的firefox快捷方式以管理员身份，复制到 /usr/bin目录
cd /usr/bin/

cd /usr/share/applications

sudo touch firefox.desktop

sudo vim firefox.desktop
