 第二章 OpenGL的基本图形

  现在我们开始在OpenGL图形界面上作一些简单的图形。

    OpenGL的基本图形有平面图形和3D图形，它们是OpenGL或计算机3D图形技术的基本构图原素。有了这些基本构图元素，原则上你可以构造出所有你能见到的3D图形（当然实际上要看你的编程技术和美术功底了）。

    OpenGL成员（库函数）的命名方式和主要类别。

      Windows下的OpenGL包含100多个库函数和四个其他类函数。每个库用前缀gl、glu、glx或aux来区分。
      OpenGL库函数还用后缀表示入口参数类型（i、f，v等）。有的函数参数类型后缀前带有数字2、3、4。2代表二维，3代表三维，4代表alpha值。有些OpenGL函数最后带一个字母v，表示函数参数可用数组来替代一系列单个参数值。

    基本图形的画法:

      OpenGL中所以图形都是由点glVertex3f()构成，基本图形画法基本形式如下。
     glBegin(类型); //连接各顶点的方式
        //图形的顶点坐标集;
     glEnd();
      顶点连接方式（构图类型）决定了构成图形的形式。

点
void Point()//画点
{ glBegin(GL_POINTS);// 单个顶点
    glVertex3f( 0.0f, 1.0f,-1.0f);  //a点
    glVertex3f(-1.0f,-1.0f, 0.0f);  //b点
    glVertex3f( 1.0f,-1.0f, 0.0f);  //c点
  glEnd();
}

线
void Line()//画线
{ glBegin(GL_LINE_LOOP); //闭合折线
    glVertex3f( 0.0f, 1.0f,-1.0f);  //a点
    glVertex3f(-1.0f,-1.0f, 0.0f);  //b点
    glVertex3f( 1.0f,-1.0f, 0.0f);  //c点
  glEnd();
}

面
void Triangle()//画面
{ glBegin(GL_POLYGON);//填充凸多边形
    glVertex3f( 0.0f, 1.0f,-1.0f);  //a点
    glVertex3f(-1.0f,-1.0f, 0.0f);  //b点
    glVertex3f( 1.0f,-1.0f, 0.0f);  //c点
  glEnd();
}
